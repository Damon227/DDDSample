## DDDSample 是基于 ASP.NET CORE2.2 的DDD示例项目。

#### 项目结构
+ WebApi
+ Application
	* ApplicationServices
	* ApplicationServices
+ Domain
	* Events
	* EventHandlers
	* Domain(领域1)
		* Models
		* DomainServices
	* Domain(领域2)
		* Models
		* DomainServices
	* DomainCore
+ Data
+ Infrastructures

#### 涉及知识
+ ASP.NET CORE 2.2
+ MediatR
+ Entityframework Core 2.2
+ AutoMapper
+ SignalR
